# MAP.md

Quick index of the Pascal4LLM breviary. Use this to decide which `.md` to read.

- `API.md` — entry point for LLMs/tooling (flow, defaults, quick index).
- `README.md` — GitHub-facing overview, project structure, and human navigation.
- `CORE-COMPATIBILITY.md` — TP7/FPC shared defaults, type sizes, directives, and compatibility matrix.
- `compilers/TP7-REFERENCE.md` — TP7-specific limits, directives, TPC options.
- `compilers/FPC-REFERENCE.md` — FPC 3.2.2 specifics and directives, with cheat sheet link.
- `compilers/MODES-COMPARISON.md` — TP vs FPC modes table.
- `topics/STRINGS-IO.md` — string handling and console I/O patterns.
- `topics/FILE-HANDLING.md` — text/typed/untyped file patterns.
- `topics/MEMORY-MODEL.md` — TP7 segmented model, stack/heap, overlays; FPC flat model.
- `topics/GRAPHICS-BGI.md` — BGI init, drivers/fonts, style tables.
- `topics/ASSEMBLY.md` — inline ASM guidance TP7/FPC.
- `topics/DIRECTIVES-PORTABILITY.md` — directives, packing/alignment, TP7 vs FPC TP mode deviations.
- `topics/PORTING-STRATEGIES.md` — TP7↔FPC porting, overlays vs unit split, ASM/calling, Exec/SwapVectors.
- `topics/DEBUGGING-PROFILING.md` — debugging with DOSBox/TD/gdb, logging, timing, assertions, lightweight profiling.
- `topics/REGRESSION-CHECKLIST.md` — regression/portability tests for strings, overflow, I/O, Crt timing, Exec.
- `examples/BASIC-PATTERNS.md` — starter TP-safe snippets.
- `reference/UNITS-REFERENCE.md` — unit catalog and TP7/FPC highlights.
- `reference/TP7-UNITS-CHEATSHEET.md` — compact TP7 API tables.
- `reference/TP7-RTL-SUMMARY.md` — broader TP7 RTL/units summary.
- `reference/FPC-UNITS-CHEATSHEET.md` — compact FPC 3.2.2 RTL/FCL/FCL-res APIs.
- `reference/FPC-RTL-FCL-SUMMARY.md` — broader FPC 3.2.2 RTL/FCL summary (token-optimized).
- `reference/ERRORS-RUNTIME.md` — runtime error mappings TP7/FPC.
- `reference/GLOSSARY.md` — terms, exceptions, TP7/FPC jargon.
- `reference/HISTORICAL-CONTEXT.md` — timeline/context.
- `legacy/TP3-REFERENCE.md` — constraints/feature set for Turbo Pascal 3.0 (no units/objects/asm blocks).
- `legacy/TP5-REFERENCE.md` — constraints/feature set for Turbo Pascal 5.0 (units present, no objects/asm blocks, smaller RTL).
- `legacy/COMPATIBILITY-MATRIX.md` — side-by-side comparison TP3 vs TP5 vs TP7 vs FPC TP mode.
- `legacy/COMMAND-LINE-CHEATSHEET.md` — per-version CLI switches/paths (TP3/TP5/TP7/FPC).
- `reference/DOSBOX-CONFIGURATION.md` — DOSBox setup for TP7 testing.
